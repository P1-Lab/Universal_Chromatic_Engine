<p style="margin:0 0 6px 0;font-size:30px;font-weight:700;">SPEC 01: The Neutral Chromatic Base (NCB)</p>

<p style="margin:0 0 18px 0;font-size:14px;color:#444;">
<strong>Standard Code:</strong> LITH-UCE-NCB-01<br>
<strong>Target Particle Size:</strong> D90 ≤ 5.0 μm<br>
<strong>Primary Function:</strong> Thermal Isolation, pH Governing, and Pigment Sequestration
</p>

<hr style="border:none;border-top:1px solid #ccc;margin:18px 0;">

<p style="margin:0 0 18px 0;">
The <strong>Neutral Chromatic Base (NCB)</strong> is a dry carrier matrix engineered to host biological pigment load-outs. Its role is to improve storage stability and process control by helping protect sensitive pigments from moisture, acidity, oxygen exposure, and premature release.
</p>

<p style="margin:0 0 14px 0;">
The functional concept of the NCB uses a three-part dry-weight ratio intended to balance carrier structure, protective lipid content, and buffering capacity.

<h3 style="font-size:17px;margin:18px 0 8px 0;">1.1 The Polysaccharide Matrix (The Core)</h3>

<p style="margin:0 0 16px 0;">
The matrix provides internal surface area for pigment adsorption or deposition. A porous structure can support concentrated color loads while maintaining a dry, free-flowing particulate form.
</p>

<h3 style="font-size:17px;margin:18px 0 8px 0;">1.2 The Hydrophobic Lamination (The Vault)</h3>

<p style="margin:0 0 16px 0;">
The lipid fraction functions as a barrier layer. When properly distributed across the carrier surface, it can reduce moisture uptake, limit oxidation pathways, and delay pigment release during storage or cold-mix processing.
</p>

<h2 style="font-size:20px;margin:24px 0 10px 0;">2. PHYSICAL &amp; THERMAL STANDARDS</h2>

<h3 style="font-size:17px;margin:18px 0 8px 0;">2.1 Thermal Latency Logic</h3>

<p style="margin:0 0 10px 0;">
Release behavior depends on the melting or softening point of the selected lipid system.
</p>

<ul style="margin:0 0 16px 22px;padding:0;">
<li><strong>State A (Inactive / Storage):</strong> Below the lipid transition range, the matrix remains relatively moisture resistant and stable.</li>
<li><strong>State B (Active / Release):</strong> Above the transition range, the barrier softens or melts, allowing pigment dispersion into the host food system.</li>
</ul>

<h3 style="font-size:17px;margin:18px 0 8px 0;">2.2 Particle Size Requirement (Sensory Zero)</h3>

<p style="margin:0 0 10px 0;"><strong>Metric:</strong> D90 ≤ 5.0 μm</p>

<p style="margin:0 0 16px 0;">
Fine particle distributions generally improve mouthfeel, dispersion speed, and visual uniformity. Larger particles may be perceptible in delicate systems such as mousses, beverages, or emulsions.
</p>

<h2 style="font-size:20px;margin:24px 0 10px 0;">3. IMPLEMENTATION AUDIT</h2>

<p style="margin:0 0 14px 0;">
Successful manufacturing requires balancing particle reduction, blending energy, and temperature control. Excess frictional heat may soften lipids prematurely or degrade heat-sensitive pigments.
</p>

<p style="margin:0 0 14px 0;">
To meet the LITH-UCE-NCB-01 concept standard, production should verify:
</p>

<ul style="margin:0 0 16px 22px;padding:0;">
<li>Consistent particle-size distribution</li>
<li>Uniform lipid dispersion</li>
<li>Low residual moisture</li>
<li>Stable pigment retention during storage</li>
<li>Predictable release during processing</li>
</ul>
